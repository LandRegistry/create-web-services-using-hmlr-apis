A guide to testing

HMLR does not currently provide a test environment for integrators using our Business Gateway APIs. Instead, we’ve provided example code to demonstrate what you should expect to see when developing your own services. You’ll find test stub examples on service API pages and example request and response code in the service API technical guides.

For additional support, contact our integration support team.

We sometimes use technical terms to describe the work we do at HMLR. For a list of commonly used terms and their definitions, visit our glossary.
