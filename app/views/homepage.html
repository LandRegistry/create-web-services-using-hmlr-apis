What's new

20 January 2025: HM Land Registry (HMLR) is developing a new service to improve the process of submitting information to HMLR.

The new Submit an application to change the Land Register service API will help to lodge applications with HMLR to update the register or create a new lease or transfer of part. This new API will replace the existing e-DRS SOAP API and is designed to help applicants get their applications right the first time.

Read more about the new Submit an application to change the Land Register service API.
