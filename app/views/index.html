Create web services using Business Gateway APIs prototype

This is a prototype of a new design for the Business Gateway developer pack. We're asking early adopters to use this version when integrating the new Submit an application to change the land register APIs.

We've focused on providing you with all the information needed to integrate the new APIs and so you'll find that some information relating to our other SOAP or REST APIs is yet to be added.

We'll be regularly updating this prototype to include all the information integrators need to build the full package of Business Gateway services.

As this is just a prototype, it's not a full service and is not production code ready. As a result, you may experience issues. It has not been accessibility tested.
